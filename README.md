# Rust To-Do Application

This repository contains a simple command-line To-Do application written in Rust. The application allows users to manage their tasks efficiently by adding, listing, and removing items from their to-do list.

## Features

- **Add Tasks**: Easily add new tasks to your to-do list.
- **List Tasks**: View all your pending tasks.
- **Remove Tasks**: Delete tasks that are completed or no longer needed.

## Getting Started

### Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (Ensure that Rust is installed on your system)

### Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/GURUSUJAN/Rust.git
2. **Navigate to the** ```todo``` **folder**
   ```bash
   cd Rust/todo
3. **Build the Application:**
   ```bash
   cargo build --release

### Usage

After building the application, you can run it using the following command:

    ./target/release/todo

Follow the on-screen prompts to add, list, or remove tasks.

Once you run the application, the following features will be available:
1. **Add a Task:**
   - To add a new task, follow the prompt and enter the task description.
   - Example:
      ```bash
   
      What would you like to add? Finish homework
      Task "Finish homework" added to the to-do list.
2. **List Tasks:**
   - To view all your pending tasks, select the option to list tasks.
   - Example:
      ```bash
   
      1. Finish homework
      2. Buy groceries
      3. Call Mom

3. **Remove a Task:**
   - To remove a completed task, choose the option to delete a task and provide the task number.
   - Example:
      ```bash
   
      Which task number would you like to remove? 2
      Task "Buy groceries" removed from the to-do list.
The application will continue to prompt you with options to add, list, or remove tasks. When you are finished, simply exit the program.

Enjoy using your simple Rust-powered To-Do application!
