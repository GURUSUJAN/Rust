# Rust To-Do Application

This repository contains a simple command-line To-Do application written in Rust. The application allows users to manage their tasks efficiently by adding, listing, and removing items from their to-do list.

## Features

- **Add Tasks**: Easily add new tasks to your to-do list.
- **List Tasks**: View all your pending tasks.
- **Remove Tasks**: Delete tasks that are completed or no longer needed.
